🚀 Quick Start — 4 Steps
1
Install dependencies
cd backend && npm install
cd frontend && npm install
2
Configure environment
cp .env.example backend/.env
Fill in MongoDB URI, JWT secrets, HuggingFace API key
3
Start the servers
cd backend && npm run dev  (Terminal 1)
cd frontend && npm run dev  (Terminal 2)
4
Open the website
http://localhost:3000 →
Sign up for a new account, then explore the dashboard
ℹ️ Reference
🔑 Keys You Need
  • MongoDB Atlas URI — cloud.mongodb.com
  • HuggingFace API key — huggingface.co
  • Adzuna keys (optional) — developer.adzuna.com
  • 3× JWT secrets — run node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
📦 Tech Stack
  • Frontend — Next.js 14 + Tailwind CSS
  • Backend — Node.js + Express
  • Database — MongoDB Atlas (free M0)
  • AI — HuggingFace free inference API
  • Jobs — Remotive + Arbeitnow (no key needed)
🚢 Deploy Free
  • Frontend → Vercel (connect GitHub repo)
  • Backend → Render (connect GitHub repo)
  • Database → MongoDB Atlas (already free)
  • See SETUP.md for full deployment guide
🛡️ Security Included
  • bcrypt passwords (12 rounds)
  • JWT in HTTP-only cookies
  • Rate limiting (5/10/20 per min)
  • MIME type file validation
  • NoSQL injection + XSS prevention
